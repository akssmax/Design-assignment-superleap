# Superleap — marketing site redesign

Next.js App Router redesign of [superleap.com](https://www.superleap.com/) using HeroUI, Framer Motion, and Lucide icons.

## Stack

- Next.js 16 + React 19 + TypeScript
- Tailwind CSS v4
- HeroUI v3 (`@heroui/react`, `@heroui/styles`)
- Framer Motion
- Lucide React

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
